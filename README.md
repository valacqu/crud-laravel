# CRUD de Productos con Laravel

Aplicación web CRUD desarrollada con Laravel que permite:

- Crear productos
- Listar productos
- Editar productos
- Eliminar productos

## Tecnologías usadas

- PHP
- Laravel
- MySQL
- HTML / CSS


## Cómo ejecutar

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
